# Mundo Aove Aceite Tacna

Site statique de vente d'huile d'olive extra vierge, producteur local à Tacna, Pérou.

## Pages
- `index.html` : Accueil / Vente
- `acerca.html` : À propos du producteur
- `contacto.html` : Contact

## Fonctionnalités
- Panier d'achat persistant en sessionStorage
- Envoi des commandes et messages via Telegram
- Suivi des visites via Google Sheets

## Structure
- `css/` : styles
- `js/` : scripts modulaires
- `data/` : données produits (JSON)
- `images/` : images

## Installation
1. Cloner le repo
2. Ouvrir avec Live Server ou tout serveur local
